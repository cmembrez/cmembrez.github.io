Project information

  • Category: Virtual Reality
  • Platform: Meta Quest 2
  • Engine and tools: Unity, Blender, GIMP
  • Project date: 16 February, 2022
  • WebGL version: link to Unity Play

Low poly adventure game from scratch

This is my first video game and development using virtual reality with the Unity game engine. I used Blender to create each low poly assets such as sword, houses and monsters, as well as GIMP to model some of the textures for the floor and walls.

This made me acquire a great global knowledge on game development methods and each chain of its applications.

This was a semester project at university. Due to hardware issue (my GPU was not supported anymore during development), the video game was presented as WebGL during the end of the semester.
